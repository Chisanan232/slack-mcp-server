# Server References

This section provides detailed documentation for all the server APIs and interfaces available in Slack MCP Server. The references are divided into three categories:

1. **Web Server** - RESTful endpoints for interacting with the server directly via HTTP
2. **MCP Server** - Programmatic interfaces for client libraries and applications  
3. **Webhook Server** - Event handling endpoints for Slack webhook integration

Choose a category below to explore the available server references:

- [Web Server](./web-server/index.mdx) - HTTP-based RESTful APIs
- [MCP Server](./mcp-server/index.mdx) - Client library interfaces
- [Webhook Server](./webhook-server/index.mdx) - Slack event handling endpoints
