# API References

This section provides detailed documentation for all the APIs available in Slack MCP Server. The APIs are divided into two categories:

1. **Web APIs** - RESTful endpoints for interacting with the server directly via HTTP
2. **MCP Server APIs** - Programmatic interfaces for client libraries and applications

Choose a category below to explore the available APIs:

- [Web APIs](./web-apis/index.mdx) - HTTP-based RESTful APIs
- [MCP Server APIs](./mcp-apis/index.mdx) - Client library interfaces
